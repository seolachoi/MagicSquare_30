# 개인별 학습 성과 정리

## 작성 가이드

- 아래 항목을 개인별로 5~10분 내 작성
- "배운 점"은 개념이 아니라 적용 사례 중심으로 작성
- "다음 적용"은 일정과 성공 기준을 포함

## 개인 성과 템플릿

| 이름 | 배운 기술/개념 | 향상된 역량 | 팀에 공유할 인사이트 | 추천 학습 자료 | 다음 적용 계획 |
|---|---|---|---|---|---|
| 예시: 홍길동 | Dual-Track TDD | 요구사항 분해 속도 향상 | Boundary/Domain 분리로 디버깅 단축 | pytest docs | 다음 스프린트에서 AC 선분해 적용 |

## 추천 학습 자료 정리

- Prompt Engineering 패턴: P-C-T-F, Few-shot, 7-Step
- 테스트 설계: pytest AAA 패턴, 경계값 분석
- 아키텍처: ECB(Entity-Control-Boundary), Clean Architecture 기본 원칙
- 품질 게이트: 커버리지 기준 수립, 회귀 테스트 운영
